Управление безопасностью через PowerShell

Внимание! Для работы этих команд PowerShell должен быть запущен от имени администратора. Изменения снижают безопасность системы.

1. Брандмауэр (Firewall)

Отключение брандмауэра

Полностью отключает сетевой экран для всех профилей (Частная, Публичная, Доменная).
Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled False

Включение брандмауэра

Возвращает настройки брандмауэра в активное состояние.
Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled True

2. Защита в реальном времени

Отключение защиты в реальном времени

Отключает основной сканер вирусов Windows Defender.
Set-MpPreference -DisableRealtimeMonitoring $true

Включение защиты в реальном времени

Включает сканирование файлов обратно.
Set-MpPreference -DisableRealtimeMonitoring $false

3. Облачная защита

Отключение облачной защиты

Отключает отправку данных и получение сигнатур из облака Microsoft (MAPS).
Set-MpPreference -MAPSReporting 0

Включение облачной защиты

Включает расширенную облачную защиту (MAPS Advanced).
Set-MpPreference -MAPSReporting 2
